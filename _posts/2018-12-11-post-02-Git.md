### 在Linux上安装
* 用yum安装  
    `$ sudo yum install git `
* 用apt-get安装  
    `$ sudo apt-get install git`
### 在Mac上安装

&emsp;&emsp;打开[ http://git-scm.com/download/mac](http://git-scm.com/download/mac) 下载并安装


### 在Window上安装
&emsp;&emsp;打开[http://windows.github.com](http://windows.github.com) 下载并安装
